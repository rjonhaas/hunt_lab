   `HL - Threat Hunt Report Template`

   `kibana/hunt_report_template.ndjson`

   `kibana/README.md`

## Portability rules

Optimize for portability across user environments.

Use these principles:

* Prefer common ECS fields
* Keep the object set minimal
* Prefer one known data view over many environment-specific references
* Avoid brittle dependencies unless you validate them
* If Lens panels make portability too fragile, use a simpler validated version rather than a fancier broken one
* A smaller template that works is better than a richer one that fails import
